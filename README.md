# Image-processor(C11):
A project which sort of feels like the spiritual successor of my matrix project. It was created in C11 and implements basic image processes such as color inversion, rotation and Sobel filtering. Upon running the program the user will be prompted with a menu that allows them to select the desired process.
## Usage notes(Very Important!):
This project supports only Bitmaps in a 4-divisible format(e.g., 1920 x 1080, 4096 x 4096 or 3840 x 2160); attempting to run anything else will result in the program failing. A sample photo is included. Please note that for the blurring it is advised to go only up to a blur factor of 15, since anything above that will take >1 minute to create the picture with that desired blur effect.
Enjoy!
